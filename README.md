# Maison Dorée — Luxury Jewelry Landing Page

A sophisticated, fully responsive landing page built for an elite gold jewelry boutique and artisan atelier. This project showcases elegant front-end engineering, fluid responsive layouts, and seamless visual transitions tailored for premium brand experiences.

---

## 🌟 Features

* **Fluid Responsive Architecture:** Completely optimized across all breakpoints (desktop, tablet, and mobile layouts) using dynamic scaling (`clamp()`) and fluid grids.
* **Staggered Image Layouts:** A beautifully crafted, staggered media presentation in the Heritage section that stacks cleanly on mobile viewports.
* **Interactive Hero Slider:** A smooth visual experience showcasing seasonal collections powered by Swiper.js.
* **Mobile-First Navigation Drawer:** A clean, hidden-offscreen slide-out mobile menu engineered entirely with CSS transitions and native JavaScript toggle states.
* **Modern Aesthetic:** Built with sophisticated typography hierarchy pairing serif headings with clean, highly scannable sans-serif body copy.

---

## 🛠️ Tech Stack & Methodologies

* **HTML5:** Semantic architecture focusing on structural accuracy and accessibility.
* **CSS3:** Custom properties (CSS variables), Flexbox, CSS Grid layout patterns, and robust media queries.
* **JavaScript:** Clean, manual DOM manipulation for interactive elements and mobile layout states.
* **Swiper.js:** For the high-end touch-enabled fade slider in the hero section.

> 💡 **Development Approach:** This project was built entirely from the ground up focusing on manual coding and raw muscle memory over cutting corners. Every selector, flex track, and media query breakpoint was written line-by-line to ensure a deep understanding of fluid web layout principles.
