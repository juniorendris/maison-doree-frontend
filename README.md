# Maison Dorée — Luxury Jewelry Landing Page

A sophisticated, fully responsive landing page built for an elite gold jewelry boutique and artisan atelier. This project showcases elegant front-end engineering, fluid responsive layouts, and seamless visual transitions tailored for premium brand experiences.

[🔗 Live Demo](https://juniorendris.github.io/maison-doree-frontend/maison.html)

---

## 🌟 Features

* **Fluid Responsive Architecture:** Completely optimized across all breakpoints using dynamic scaling (`clamp()`) and fluid grids.
* **Staggered Image Layouts:** A beautifully crafted media presentation in the Heritage section that stacks cleanly on mobile viewports.
* **Interactive Hero Slider:** A smooth visual experience showcasing seasonal collections powered by Swiper.js.
* **Mobile-First Navigation Drawer:** A clean, hidden-offscreen slide-out mobile menu engineered with CSS transitions and native JavaScript toggle states.

---

## 🛠️ Tech Stack & Methodologies

* **HTML5 & CSS3:** Semantic markup, custom properties (CSS variables), Flexbox, CSS Grid, and custom media queries.
* **JavaScript (ES6+):** Clean, manual DOM manipulation for layout interactions.
* **Swiper.js:** Touch-enabled fade slider integration.

> 💡 **Development Approach:** This project was built entirely from the ground up focusing on manual coding and raw muscle memory over cutting corners. Every selector, flex track, and breakpoint was written line-by-line to master fluid web layout design.
